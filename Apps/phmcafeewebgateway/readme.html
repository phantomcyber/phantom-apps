Setup

In order to set up an asset, you will need to first do the following on McAfee Web Gateway:
  1. Enable the REST interface (Click HERE to go to McAfee documentation)
  2. Give permission to access (Click HERE to go to McAfee documentation)

Note: It is important to create a separate account for Splunk> Phantom to use since a user cannot log in more than once at a time.

Block/Unblock Actions

block actions (block url|ip|domain) only add items to a provided list. It is up to the McAfee Web Gateway Administrators to manage the rules used for using that list in the correct rule. unblock actions (unblock url|ip|domain) only remove items from a provided list. It is up to the McAfee Web Gateway Administrators to manage the rules used for using that list in the correct rule.
