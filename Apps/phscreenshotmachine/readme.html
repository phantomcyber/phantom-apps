Asset Configuration

By using the cache_limit configuration parameter you can manage how old(in days) cached images do you accept. Allowed values are 0-14. A zero value means always download fresh screenshots. If you need a shorter interval than one day, you can use decimal numbers in the parameter, e.g. cacheLimit=0.041666 means: use image from cache only if it is not older than one HOUR (1/24=0.041666).
