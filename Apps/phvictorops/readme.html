To get started with the VictorOps API to integrate with Phantom, log into the VictorOps web portal and go to Integrations then API. From here you can retrieve your API ID and have the opportunity to create your API Keys. Note: Only VictorOps admin users can create API Keys. The documentation for the VictorOps API allows you to try out the different calls in real time. To get started read the API Documentation.

Phantom integrates with the VictorOps REST Endpoint to trigger, update, or resolve incidents in VictorOps.
