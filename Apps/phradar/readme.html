Asset Configuration

Configuration that must be set are under the "Asset Settings" tab. Below lists some details for each configuration setting:

  • Radar API Bearer Token - A valid Radar bearer token with privileges for reading and writing radar incidents and notes.
  • Radar API URL - defaults to the production URL, but can be set to a test or staging URL.
  • Incident Group ID - must be set to the incident group of the user associated with the bearer token.
  • Time Zone - The timezone to be associated with the asset. Used to set the Radar incident 'discovered' time.

Environment Variables

Under the 'Advanced' section of the asset settings, set environment variables to configure the app settings:

ALLOW_SELF_SIGNED_CERTS - Should be set to '1' to turn off verification of SSL Certificates during action requests.
